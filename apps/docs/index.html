Reusable components for Alpine.js

Components for Alpine.js

Define reusable template components in their own files and render them anywhere with a single directive — no build step required.

components/Counter.js 
// Define your component once
export default {
  template: `
    <div x-data="__self__">
      <p>Count: <span x-text="count"></span></p>
      <button @click="increment">+</button>
    </div>
  `,
  data: () => ({ count: 0, increment() { this.count++ } })
}
index.html 
<!-- Use it anywhere -->
<div x-component="'Counter'"></div>

Getting Started

Up and running in under 2 minutes.

1

Install

terminal 
# runtime
npm install alpine-components

# optional: vite plugin for auto-importing components
npm install -D vite-plugin-alpine-components
2

Register the plugin

main.js 
import Alpine from 'alpinejs'
import AlpineComponents from 'alpine-components'
// if using the vite plugin:
import 'virtual:alpine-components/components'

Alpine.plugin(AlpineComponents)
Alpine.start()
3

Create a component

src/components/Counter.js 
export default {
  template: `
    <div x-data="__self__">
      <p>Count: <span x-text="count"></span></p>
      <button @click="increment">+</button>
    </div>
  `,
  data: () => ({
    count: 0,
    increment() { this.count++ }
  })
}

The Vite plugin auto-registers anything in src/components/. Without the plugin, call registerComponent('Counter', Counter) manually before Alpine.start().

4

Use it in your HTML

index.html 
<div x-component="'Counter'"></div>
